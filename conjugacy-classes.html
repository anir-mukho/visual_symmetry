GROUP THEORY BY EXAMPLE · Module 6 · Conjugation

The same move, seen from a different seat.

Turn the cube 90° about the front face; turn it 90° about the top face. Different elements of the group — but not really different moves. One is the other, watched by someone who has walked a quarter of the way round the table. Making that sentence precise gives us g x g⁻¹, and it sorts every group into families called conjugacy classes. This is the last idea we need before the linear groups.

01 — THE CHANGE-OF-VIEWPOINT FORMULA

Move there, act, move back

Let x be the move you know how to do, and let g be some other element of the group — think of g as "carrying the object into a new position." Then

xg  =  g x g⁻¹

reads right-to-left as a three-step recipe: undo g, then do x, then redo g. It is the same move x, performed in the position that g puts you in. Two elements related this way are called conjugate, written x ∼ gxg⁻¹.

The one line that explains everything. If x fixes a point p, then gxg⁻¹ fixes the point g(p). Check it: (gxg⁻¹)(gp) = gx(p) = g(p). So conjugating does not change what the move does — it only transports the axis, the mirror, the fixed points, along with the object.

Watch it on a triangle. The reflection s₁ holds vertex 1 still and swaps the other two. The rotation r = (1 2 3) carries vertex 1 to vertex 2. Conjugating s₁ by r must therefore give the reflection that holds vertex 2 still — the mirror line rides along with the vertex it passes through.

The mirror is not destroyed and rebuilt — it is carried. Vertex 1 goes to vertex 2, so the axis through 1 goes to the axis through 2.

All three reflections of the triangle are conjugate to one another; the two rotations r, r² are conjugate to each other; the identity is alone. Those are the conjugacy classes of S₃ ≅ D₃, of sizes 1 + 2 + 3 = 6.

02 — THE CLEANEST EXAMPLE

In Sn, conjugation is just relabelling

For permutations the rule needs no thought at all. To conjugate a cycle by g, apply g to every symbol inside it:

g (a b c) g⁻¹  =  ( g(a)   g(b)   g(c) )

Take x = (1 2 3) and g = (1 4), the swap of 1 and 4. Then gxg⁻¹ = (4 2 3). The picture of the permutation — one triangle-shaped cycle, one fixed dot — is untouched; only the names on the dots have moved.

Same diagram, new nameplates. Conjugation cannot change the cycle shape — and nothing else survives it either.

So in Sₙ: two permutations are conjugate exactly when they have the same cycle shape. The classes are therefore counted by the partitions of n — which is why S₄ had the five families 1 + 6 + 3 + 8 + 6 = 24 we met in Module 4, and why the cube's rotations sorted themselves into exactly five geometric types.

03 — CLASSES CUT THE GROUP INTO BLOCKS

Every element sits in exactly one class

"Being conjugate" is an equivalence relation: x = exe⁻¹ (reflexive); if y = gxg⁻¹ then x = g⁻¹y g (symmetric); and conjugating twice, by g then h, is conjugating once by hg (transitive). So the classes partition the group — no overlaps, nothing left over. Three examples, drawn to scale:

Each bar is a whole group; each block is one conjugacy class. Class sizes always add up to |G| — and each one divides |G|.

Two facts fall straight out of the picture. First, the identity is always alone in its class, since geg⁻¹ = e. Second, in an abelian group gxg⁻¹ = xgg⁻¹ = x for every g, so every class is a single element. Conjugacy is a measure of how badly a group fails to commute — and it is invisible in ℤₙ and in SO(2). That is exactly why the story only becomes interesting when we reach SU(2) and SO(3).

Why the sizes divide |G|

Fix x. The elements that fail to move it — those with gxg⁻¹ = x, i.e. gx = xg — form a subgroup, the centralizer C(x). Two elements g, h produce the same conjugate of x precisely when they differ by something in C(x). So the conjugates of x correspond one-to-one with the cosets of C(x):

| class of x |  =  |G| / |C(x)|

Example, in S₄: the transposition (1 2) commutes with e, (1 2), (3 4), (1 2)(3 4) — a centralizer of size 4 — so its class has 24/4 = 6 members. The table of transpositions has exactly six entries. This is the orbit–stabiliser theorem in disguise: the group acts on itself by conjugation, and the classes are the orbits.

04 — WHAT SURVIVES CONJUGATION

The class is what is left when you forget the viewpoint

Anything that a change of seat cannot affect is constant on a conjugacy class. Such a quantity is called a class function, and the list is short and useful:

any groupg x g⁻¹the order of the elementa family of same-order moves

rotation groupsmove the axisthe angle of turnall axes, one fixed angle
matrix groupschange of basistrace, determinant, eigenvaluesa similarity class

That last row is the punchline for everything ahead. In a linear group, g X g⁻¹ is exactly what happens to the matrix X when you rewrite it in a new basis — the map is the same map; only the coordinates changed. Conjugate matrices are similar matrices, and similar matrices share their trace, determinant and eigenvalues. Conjugacy classes are basis-free facts about a linear map.

Geometrically, in SO(3): conjugating a rotation swings its axis somewhere else on the sphere but never touches the angle. So a conjugacy class of SO(3) is precisely "turn by θ, about any axis you like" — one class for each angle θ ∈ [0, π].

Conjugation slides the axis over the sphere and leaves the angle alone. The class is labelled by a single number.
Where this is going. In SU(2) the same thing happens, and it becomes fully visible. Every element is g = cos(θ/2)·I + i sin(θ/2)·(n·σ), its trace is 2cos(θ/2), and conjugation moves the unit vector n anywhere on the sphere while fixing θ. SU(2) is the 3-sphere S³, and its conjugacy classes are the 2-spheres of constant θ — latitude by latitude, poles at ±I. That is the next page.

Normal subgroups. A subgroup N is normal exactly when gNg⁻¹ = N — that is, when N is a union of whole conjugacy classes. Look back at the bar for A₄: the blocks 1 + 3 combine to a normal subgroup of order 4; you cannot slice a class in half. This is how one finds every normal subgroup, and why A₅ — whose class sizes 1, 15, 20, 12, 12 admit no sub-sum dividing 60 except 1 — is simple, and hence why the quintic has no formula.

Characters. A representation sends group elements to matrices; its character is the trace. Since tr(gXg⁻¹) = tr(X), characters are constant on classes — so a character table needs only one column per class, and the number of irreducible representations equals the number of conjugacy classes. Every class count in this course — 4 for A₄, 5 for S₄ — has been quietly telling you the width of a table in Part 2.
